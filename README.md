# DropAir BOT
DropAir BOT

Register Here : [DropAir](https://dropair.io/?ref=9S9TD6)

## Fitur

  - Auto Get Account Information
  - Auto Claim Daily Check-In
  - Auto Complete Available Tasks
  - Multi Accounts

## Prasyarat

Pastikan Anda telah menginstal Python3.9 dan PIP.

## Instalasi

1. **Kloning repositori:**
   ```bash
   git clone https://github.com/vonssy/DropAir-BOT.git
   ```
   ```bash
   cd DropAir-BOT
   ```

2. **Instal Requirements:**
   ```bash
   pip install -r requirements.txt #or pip3 install -r requirements.txt
   ```

## Konfigurasi

- **tokens.txt:** Anda akan menemukan file `tokens.txt` di dalam direktori proyek. Pastikan `tokens.txt` berisi data yang sesuai dengan format yang diharapkan oleh skrip. Berikut adalah contoh format file:

  ```bash
    eyjxxxxx1
    eyjxxxxx2
  ```

## Jalankan

```bash
python bot.py #or python3 bot.py
```
